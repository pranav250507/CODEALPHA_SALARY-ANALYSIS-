# Global Data Jobs Market Analysis

## Project Overview

This project analyzes a dataset of global data-related job postings to uncover insights into salary trends, hiring demand, and the most sought-after technical skills. The analysis was performed using Python for data cleaning and exploratory data analysis (EDA), while Tableau was used to create professional visualizations.

The objective of this project is to answer key business questions such as:
- Which data job roles offer the highest salaries?
- Which companies and countries provide the best career opportunities?
- Which technical skills are most in demand among employers?

---

## Project Objectives

### Salary Analysis
- Analyze salary trends across different job roles.
- Identify the highest paying companies.
- Compare salaries across countries.
- Understand salary distribution within the data job market.

### Hiring Trends Analysis
- Identify the most in-demand job roles.
- Analyze hiring distribution across companies.
- Explore hiring trends across different countries.

### Skills Analysis
- Discover the most frequently requested technical skills.
- Analyze the relationship between skills and salary.
- Identify valuable technologies for data professionals.

# Salary Analysis

### Objective

Analyze salary patterns across different data job roles, companies, and countries.

### Visualizations

- Average Salary by Job Title
- Salary by Country
- Top Paying Companies
- Salary Distribution

### Key Insights

- Machine Learning Engineers receive the highest average salaries.
- Salary levels vary considerably across countries.
- Specialized technical roles generally receive higher compensation.
- Top-paying companies consistently offer salaries above the market average.

---

# Hiring Trends Analysis

### Objective

Analyze hiring demand within the global data job market.

### Visualizations

- Most In-Demand Job Roles
- Top Hiring Companies
- Jobs by Country
- Hiring Distribution

### Key Insights

- Data Analyst is one of the most frequently advertised positions.
- Hiring demand is concentrated among a relatively small number of companies.
- The United States contributes the highest number of job postings.
- Demand for data professionals remains strong across multiple industries.
---

# Skills Analysis

### Objective

Understand which technical skills employers value the most.

### Visualizations

- Top Technical Skills
- Salary by Skill
- Skill Demand Comparison

### Key Insights

- SQL and Python are the most frequently requested technical skills.
- Cloud technologies such as AWS and Azure are associated with higher salaries.
- Tableau and Power BI remain important business intelligence tools.
- Advanced machine learning skills command premium salaries.

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Global-Data-Jobs-Market-Analysis.git
```

Navigate to the project directory:

```bash
cd Global-Data-Jobs-Market-Analysis
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Launch JupyterLab:

```bash
jupyter lab
```

Open `salary_analysis.ipynb` and run the notebook.

## Key Takeaways

- Data-related careers continue to offer competitive salaries across industries.
- Data Analyst and Data Scientist remain among the most in-demand roles.
- Python and SQL are the foundational skills required for most data careers.
- Cloud technologies and machine learning skills are associated with higher-paying positions.
- Geographic location has a significant impact on salary levels.
